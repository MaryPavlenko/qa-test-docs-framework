# Responsive UI Checklist — Swag Labs

## Desktop View

- [ ] Open login page on desktop resolution → Layout is displayed correctly
- [ ] Open product catalogue on desktop resolution → Product cards are aligned correctly
- [ ] Open cart page on desktop resolution → Cart items are displayed correctly
- [ ] Open checkout page on desktop resolution → Form fields and buttons are displayed correctly

## Tablet View

- [ ] Open login page on tablet resolution → Layout is displayed correctly
- [ ] Open product catalogue on tablet resolution → Product cards adapt to available screen width
- [ ] Open cart page on tablet resolution → Cart content remains readable
- [ ] Open checkout page on tablet resolution → Form fields remain usable

## Mobile View

- [ ] Open login page on mobile resolution → Login form fits the screen
- [ ] Open product catalogue on mobile resolution → Product cards are readable and usable
- [ ] Open product details on mobile resolution → Product image, description and price are displayed correctly
- [ ] Open cart page on mobile resolution → Cart items do not overlap
- [ ] Open checkout page on mobile resolution → Input fields and buttons fit the screen
- [ ] Complete checkout on mobile resolution → Flow remains usable

## UI Behaviour

- [ ] Verify text does not overlap buttons
- [ ] Verify product images scale correctly
- [ ] Verify buttons remain clickable on small screens
- [ ] Verify cart icon remains visible
- [ ] Verify menu opens correctly on smaller screens
- [ ] Verify no unexpected horizontal scrolling appears
- [ ] Verify validation messages are readable on small screens

## Browser DevTools Checks

- [ ] Test layout using Chrome DevTools Device Toolbar
- [ ] Check common resolutions: 320px, 375px, 768px, 1024px, 1440px
- [ ] Check portrait and landscape orientation where applicable
