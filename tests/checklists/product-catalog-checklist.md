# Product Catalogue Checklist — Swag Labs

## Positive Checks

- [ ] Log in with valid credentials → Product catalogue page is displayed
- [ ] Open product catalogue → Product list is visible
- [ ] Check each product card → Product name, image, description and price are displayed
- [ ] Click product image/name → Product details page is opened
- [ ] Click “Add to cart” on a product → Product is added to cart
- [ ] Add product to cart → Button changes from “Add to cart” to “Remove”
- [ ] Click “Remove” → Product is removed from cart
- [ ] Open sorting dropdown → Sorting options are displayed
- [ ] Select “Name A to Z” → Products are sorted alphabetically ascending
- [ ] Select “Name Z to A” → Products are sorted alphabetically descending
- [ ] Select “Price low to high” → Products are sorted by price ascending
- [ ] Select “Price high to low” → Products are sorted by price descending

## Negative / Edge Checks

- [ ] Refresh product catalogue page → Product list remains available
- [ ] Open catalogue with poor internet connection → Page loading state / browser behaviour is handled correctly
- [ ] Add several products to cart → Cart counter displays correct number of products
- [ ] Remove all added products → Cart counter disappears

## UI Checks

- [ ] Verify product images are loaded correctly
- [ ] Verify product prices are displayed in the same format
- [ ] Verify product cards are aligned correctly
- [ ] Verify no text overlaps product images or buttons
